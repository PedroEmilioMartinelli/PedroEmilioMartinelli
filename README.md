## 🚀 Projeto em Destaque

<div align="center">

[![LogSentinel](https://github-readme-stats.vercel.app/api/pin/?username=PedroEmilioMartinelli&repo=LogSentinel-Mini-SIEM-SOC-Analyzer-&theme=dark&hide_border=true)](https://github.com/PedroEmilioMartinelli/LogSentinel-Mini-SIEM-SOC-Analyzer-)

</div>

**LogSentinel** é um Mini SIEM desenvolvido do zero em Python, capaz de:
- Ler e parsear logs de autenticação SSH e requisições HTTP em tempo real
- Detectar automaticamente brute force, comportamento suspeito e padrões de DDoS
- Correlacionar eventos e bloquear IPs maliciosos
- Exibir tudo em um dashboard web com atualização automática
